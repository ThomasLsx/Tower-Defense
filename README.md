# Tower-Defense

Projet de Lilian Briaut et Thomas Lesieux

## Instructions pour lancer le projet

1. Clonez le dépôt :
   ```
   git clone https://github.com/ThomasLsx/Tower-Defense.git
   ```
2. Ouvrez le fichier `Tower Defense.sln` avec Visual Studio.
3. Cliquez sur la flèche "Run" (Démarrer) pour exécuter le projet.
